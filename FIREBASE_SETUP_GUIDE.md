# データベース(Firestore)の作成手順

送っていただいた画像を確認しました！
現在開いているのは **「Realtime Database」** ですが、このアプリで使用するのは **「Firestore Database」** です。
（名前が似ていますが、別の種類のデータベースです）

以下の手順で、正しいデータベースを作成してください。これを行わないと、保存機能は動きません。

## 手順1: Firestore Databaseを作る

1. 左側のメニューにある **「構築 (Build)」** をクリックして、メニューを展開してください。
2. その中に出てくる **「Firestore Database」** をクリックします。
   - ※もしメニューにない場合、画面上の「すべてのプロダクト」から探してください。
3. 画面中央にある **「データベースの作成 (Create database)」** ボタンをクリックします。
4. **設定ウィザード**が表示されます：
   - **ロケーション**: `asia-northeast1 (Tokyo)` （またはデフォルトのまま）を選択して「次へ」。
   - **セキュリティルール**: 必ず **「テストモードで開始する (Start in test mode)」** を選択してください！
     - これを選ぶと、面倒なルール設定なしで、すぐにアプリから保存できるようになります。
5. **「作成 (Create)」** をクリックします。

## 手順2: Storageを作る（まだの場合）

写真の保存場所も同様に作成が必要です。

1. 左メニューの **「Storage」** をクリックします。
2. もし **「始める (Get started)」** ボタンが表示されていたら、クリックして作成します。
3. こちらも **「テストモードで開始する」** を選択して作成してください。
